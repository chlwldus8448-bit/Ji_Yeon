Project Tools

요미&요시는 “행복한 꾸러기들”이라는 메시지를 바탕으로
아이들에게 긍정적인 감정과 편안한 에너지를 전달하기 위해 디자인된 캐릭터입니다.
둥글고 부드러운 선, 파스텔 톤 컬러, 과장되지 않은 표정을 사용하여
따뜻함·안정감·친근함이 자연스럽게 느껴지도록 설계했습니다.
각 캐릭터는 고유한 성격과 감정 표현을 통해 스토리와 굿즈로 확장되는 IP 구조를 갖습니다.
