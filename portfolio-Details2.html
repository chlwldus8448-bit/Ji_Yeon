Project Tools

MOAV는 작은 오브제들이 모여 하나의 따뜻한 공간을 완성한다는 철학을 담은 라이프스타일 브랜드로
브랜드의 감성과 결을 시각적으로 경험할 수 있도록 웹 인터페이스를 설계했습니다.
본 프로젝트는 MOAV의 따뜻한 분위기와 공간적 무드를 사용자에게 자연스럽게 전달하기 위해
**반응형 웹 디자인(PC · Tablet · Mobile)**으로 제작되었습니다.
컬렉션, 라이프스타일 인스피레이션, 제품 페이지로 이어지는 구조를 통해
브랜드 스토리와 조형적 따뜻함을 동시에 보여주는 웹 경험을 구현했습니다.
